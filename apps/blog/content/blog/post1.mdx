# 첫 번째 MDX 블로그 포스트

안녕하세요! 이것은 **Next.js App Router**와 **MDX**를 사용하여 작성된 첫 번째 블로그 포스트입니다.

## MDX란 무엇인가요?

MDX는 마크다운과 JSX를 결합한 형식으로, 마크다운 내에서 React 컴포넌트를 사용할 수 있게 해줍니다.

```typescript
// 예제 코드
const greeting = "안녕하세요!"
console.log(greeting)
```

### 커스텀 컴포넌트 사용하기

아래는 커스텀 이미지 컴포넌트를 사용하는 예시입니다:

<Image
	src="/placeholder-image.jpg"
	alt="예시 이미지"
	width={800}
	height={400}
/>

## MDX의 장점

1. **마크다운의 간결함** - 콘텐츠를 빠르게 작성할 수 있습니다.
2. **React 컴포넌트 사용** - 동적인 요소를 추가할 수 있습니다.
3. **타입스크립트 지원** - 타입 안전성을 보장합니다.

> MDX는 기술 블로그를 작성하는 데 탁월한 선택입니다!
